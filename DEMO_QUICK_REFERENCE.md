## ⚡ Super Quick Start

```bash
# 1. Install & Setup (2 minutes)
npm install
npx playwright install chromium

# 2. Start PLUTO (30 seconds)
npm run dev
# → Open http://localhost:3000

# 3. Demo Ready! 🚀
```

## 🚨 Troubleshooting

| Issue | Quick Fix |
|-------|-----------|
| Port 3000 busy | `npx kill-port 3000` |
| Playwright issues | `npx playwright install --force chromium` |
| Extension not loading | Enable Developer mode in Chrome |
| API errors | Check `.env` file has Groq API key |
